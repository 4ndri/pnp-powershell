## SYNOPSIS

## SYNTAX

```powershell
Get-PnPSiteCollectionAppCatalogs
 [<CommonParameters>]
```

## DESCRIPTION
Returns site collection scoped app catalogs

## EXAMPLES

### EXAMPLE 1
