# New-PnPSiteTemplate

## SYNOPSIS
Creates a new provisioning template object

## SYNTAX

```
New-PnPSiteTemplate [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION

## EXAMPLES

### EXAMPLE 1
```powershell
$template = New-PnPSiteTemplate
```

Creates a new instance of a site template object.
